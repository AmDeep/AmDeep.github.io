# Amardeep Singh Portfolio

This is a React + Tailwind + Framer Motion portfolio site for Amardeep Singh.

## Deploy via GitHub Web UI (no CLI)
1. Create a new public repo on GitHub (e.g., `amardeep-portfolio`).
2. Click **Add file → Upload files**, drag all the contents of this ZIP, and **Commit changes**.
3. Go to **Settings → Pages → Build and deployment → Source**, select **GitHub Actions**. GitHub will automatically detect the workflow and deploy the site.
4. Visit the live site (check Pages URL in Settings).

To change text or add new experience/projects: edit `/src/App.jsx` directly from the web editor.
